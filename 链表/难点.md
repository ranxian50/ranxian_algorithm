# 2. 环形链表判断（快慢指针）
**思路：**
- 快指针一次走 2 步，慢指针一次走 1 步
- 如果有环，快慢指针一定会相遇
- 若无环，快指针会走到 NULL

```cpp
bool hasCycle(ListNode *head) {
    ListNode *slow = head;
    ListNode *fast = head;
    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) return true;
    }
    return false;
}
```

# 3. 链表中间节点（快慢指针）
**思路：**
- 快指针走 2 步，慢指针走 1 步
- 快指针到末尾时，慢指针就在中间

```cpp
ListNode* middleNode(ListNode* head) {
    ListNode *slow = head;
    ListNode *fast = head;
    while (fast != NULL && fast->next != NULL) {
        slow = slow->next;
        fast = fast->next->next;
    }
    return slow;
}
```

# 4. 合并两个有序链表
**思路：**
- 新建虚拟头结点，用一个指针尾插
- 谁小就先接谁
- 最后把剩下的链表接上

```cpp
ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
    ListNode dummy(0);
    ListNode* cur = &dummy;
    while (l1 && l2) {
        if (l1->val < l2->val) {
            cur->next = l1;
            l1 = l1->next;
        } else {
            cur->next = l2;
            l2 = l2->next;
        }
        cur = cur->next;
    }
    cur->next = l1 ? l1 : l2;
    return dummy.next;
}
```

# 5. 删除倒数第 N 个节点
**思路：**
- 快慢指针，快指针先走 n 步
- 然后一起走，快到尾时，慢就在倒数 n+1 位置
- 直接删除慢指针的下一个

```cpp
ListNode* removeNthFromEnd(ListNode* head, int n) {
    ListNode dummy(0);
    dummy.next = head;
    ListNode *slow = &dummy, *fast = &dummy;
    // 快指针先走 n 步
    for (int i = 0; i <= n; ++i) fast = fast->next;
    while (fast) {
        slow = slow->next;
        fast = fast->next;
    }
    ListNode* tmp = slow->next;
    slow->next = slow->next->next;
    delete tmp;
    return dummy.next;
}
```

# 6.1 链表相交
**思路：**
- 两个指针分别走 A+B 和 B+A
- 有相交一定会相遇，无相交则同时为 NULL

```cpp
ListNode *getIntersectionNode(ListNode *headA, ListNode *headB) {
    ListNode *pA = headA, *pB = headB;
    while (pA != pB) {
        pA = pA ? pA->next : headB;
        pB = pB ? pB->next : headA;
    }
    return pA;
}
```

# 6.2 回文链表判断
**思路：**
1. 找中间节点
2. 反转后半段
3. 前后两段逐一比较

```cpp
// 反转链表（辅助函数）
ListNode* reverseList(ListNode* head) {
    ListNode *prev = NULL, *cur = head;
    while (cur) {
        ListNode* next = cur->next;
        cur->next = prev;
        prev = cur;
        cur = next;
    }
    return prev;
}

bool isPalindrome(ListNode* head) {
    if (!head || !head->next) return true;
    // 找中点
    ListNode *slow = head, *fast = head;
    while (fast->next && fast->next->next) {
        slow = slow->next;
        fast = fast->next->next;
    }
    // 反转后半段
    ListNode* second = reverseList(slow->next);
    // 比较
    ListNode *p1 = head, *p2 = second;
    bool ok = true;
    while (ok && p2) {
        if (p1->val != p2->val) ok = false;
        p1 = p1->next;
        p2 = p2->next;
    }
    // 恢复链表（可选）
    slow->next = reverseList(second);
    return ok;
}
```
