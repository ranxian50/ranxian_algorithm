# 链表（Linked List）全面详细讲解（C++ 实现）
链表是**最基础、最重要**的线性数据结构之一，它和数组并列是编程中必须掌握的核心结构。我会从**概念、分类、结构、操作、优缺点、代码实现**全方位带你吃透链表。

## 一、什么是链表？
### 1. 核心定义
链表是一种**非连续、非顺序**的存储结构，数据元素的逻辑顺序通过**指针（地址）** 串联实现。

简单理解：
- 数组：像一排连续的房间，门牌号连续，找起来快，但插入删除麻烦
- 链表：像一串珠子，每个珠子只知道**下一个珠子在哪**，空间不连续，插入删除极方便

### 2. 链表的基本单元：节点
链表由一个个**节点（Node）** 组成，每个节点包含两部分：
1. **数据域**：存储真实数据
2. **指针域**：存储**下一个节点的地址**

## 二、链表的分类
最常用的 3 种链表：
1. **单链表**：每个节点只有一个指向下一节点的指针
2. **双链表**：每个节点有前驱、后继两个指针
3. **循环链表**：尾节点指针指向头节点，形成闭环

本文重点讲解**单链表 + 双链表**（面试/工程最常用）。

## 三、单链表（Singly Linked List）
### 1. 结构示意图
```
头节点 -> 节点1 -> 节点2 -> 节点3 -> NULL
```
- 只能**从头向后遍历**，不能反向查找
- 最后一个节点的指针指向 `NULL`

### 2. C++ 节点定义
```cpp
// 单链表节点结构
struct ListNode {
    int val;          // 数据域
    ListNode *next;   // 指针域：指向下一个节点
    // 构造函数
    ListNode(int x) : val(x), next(nullptr) {}
};
```

### 3. 单链表核心操作（C++ 完整实现）
包含：**创建、遍历、头插、尾插、指定位置插入、删除、查找、销毁**

```cpp
#include <iostream>
using namespace std;

// 单链表节点定义
struct ListNode {
    int val;
    ListNode *next;
    ListNode(int x) : val(x), next(nullptr) {}
};

class SingleLinkedList {
private:
    ListNode *head; // 头节点指针
public:
    // 构造函数：初始化空链表
    SingleLinkedList() {
        head = nullptr;
    }

    // 1. 尾部插入节点
    void addAtTail(int val) {
        ListNode *newNode = new ListNode(val);
        // 如果链表为空，直接作为头节点
        if (head == nullptr) {
            head = newNode;
            return;
        }
        // 找到最后一个节点
        ListNode *cur = head;
        while (cur->next != nullptr) {
            cur = cur->next;
        }
        cur->next = newNode;
    }

    // 2. 头部插入节点
    void addAtHead(int val) {
        ListNode *newNode = new ListNode(val);
        newNode->next = head; // 新节点指向原头节点
        head = newNode;      // 头节点更新为新节点
    }

    // 3. 遍历输出链表
    void traverse() {
        ListNode *cur = head;
        while (cur != nullptr) {
            cout << cur->val << " -> ";
            cur = cur->next;
        }
        cout << "NULL" << endl;
    }

    // 4. 获取第 index 个节点的值（index 从 0 开始）
    int get(int index) {
        if (index < 0) return -1;
        ListNode *cur = head;
        int i = 0;
        while (cur != nullptr && i < index) {
            cur = cur->next;
            i++;
        }
        // 找到节点返回值，否则返回 -1
        return cur ? cur->val : -1;
    }

    // 5. 在指定位置插入节点
    bool addAtIndex(int index, int val) {
        if (index < 0) return false;
        // 插入头部
        if (index == 0) {
            addAtHead(val);
            return true;
        }

        ListNode *cur = head;
        int i = 0;
        // 找到第 index-1 个节点
        while (cur != nullptr && i < index - 1) {
            cur = cur->next;
            i++;
        }
        if (cur == nullptr) return false; // 越界

        ListNode *newNode = new ListNode(val);
        newNode->next = cur->next;
        cur->next = newNode;
        return true;
    }

    // 6. 删除指定位置节点
    bool deleteAtIndex(int index) {
        if (head == nullptr || index < 0) return false;

        ListNode *temp = nullptr;
        // 删除头节点
        if (index == 0) {
            temp = head;
            head = head->next;
            delete temp;
            return true;
        }

        ListNode *cur = head;
        int i = 0;
        // 找到待删除节点的前驱
        while (cur->next != nullptr && i < index - 1) {
            cur = cur->next;
            i++;
        }
        if (cur->next == nullptr) return false; // 越界

        temp = cur->next;
        cur->next = temp->next;
        delete temp;
        return true;
    }

    // 7. 销毁链表（防止内存泄漏）
    ~SingleLinkedList() {
        ListNode *cur = head;
        while (cur != nullptr) {
            ListNode *temp = cur;
            cur = cur->next;
            delete temp;
        }
        head = nullptr;
    }
};

// 测试
int main() {
    SingleLinkedList list;
    list.addAtTail(10);
    list.addAtTail(20);
    list.addAtHead(5);
    list.addAtIndex(2, 15);

    cout << "链表内容：";
    list.traverse(); // 5 -> 10 -> 15 -> 20 -> NULL

    cout << "索引 2 的值：" << list.get(2) << endl; // 15

    list.deleteAtIndex(2);
    cout << "删除索引 2 后：";
    list.traverse(); // 5 -> 10 -> 20 -> NULL

    return 0;
}
```

## 四、双链表（Doubly Linked List）
### 1. 结构示意图
```
NULL <-> 节点1 <-> 节点2 <-> 节点3 <-> NULL
```
- 每个节点有 **prev（前驱）** 和 **next（后继）** 两个指针
- 支持**正向 + 反向**遍历
- 插入/删除效率比单链表更高（不用找前驱）

### 2. C++ 节点定义
```cpp
struct DoubleListNode {
    int val;
    DoubleListNode *prev; // 前驱指针
    DoubleListNode *next; // 后继指针
    DoubleListNode(int x) : val(x), prev(nullptr), next(nullptr) {}
};
```

### 3. 双链表核心操作（简化版）
```cpp
#include <iostream>
using namespace std;

struct DoubleListNode {
    int val;
    DoubleListNode *prev;
    DoubleListNode *next;
    DoubleListNode(int x) : val(x), prev(nullptr), next(nullptr) {}
};

class DoubleLinkedList {
private:
    DoubleListNode *head;
    DoubleListNode *tail; // 尾指针，方便尾插
public:
    DoubleLinkedList() {
        head = nullptr;
        tail = nullptr;
    }

    // 尾插
    void addAtTail(int val) {
        DoubleListNode *newNode = new DoubleListNode(val);
        if (head == nullptr) {
            head = newNode;
            tail = newNode;
            return;
        }
        tail->next = newNode;
        newNode->prev = tail;
        tail = newNode;
    }

    // 头插
    void addAtHead(int val) {
        DoubleListNode *newNode = new DoubleListNode(val);
        if (head == nullptr) {
            head = newNode;
            tail = newNode;
            return;
        }
        newNode->next = head;
        head->prev = newNode;
        head = newNode;
    }

    // 正向遍历
    void traverseForward() {
        DoubleListNode *cur = head;
        while (cur != nullptr) {
            cout << cur->val << " <-> ";
            cur = cur->next;
        }
        cout << "NULL" << endl;
    }

    // 反向遍历
    void traverseBackward() {
        DoubleListNode *cur = tail;
        while (cur != nullptr) {
            cout << cur->val << " <-> ";
            cur = cur->prev;
        }
        cout << "NULL" << endl;
    }

    // 析构函数
    ~DoubleLinkedList() {
        DoubleListNode *cur = head;
        while (cur != nullptr) {
            DoubleListNode *temp = cur;
            cur = cur->next;
            delete temp;
        }
        head = tail = nullptr;
    }
};

// 测试
int main() {
    DoubleLinkedList dList;
    dList.addAtTail(10);
    dList.addAtTail(20);
    dList.addAtHead(5);

    cout << "正向遍历：";
    dList.traverseForward();

    cout << "反向遍历：";
    dList.traverseBackward();
    return 0;
}
```

## 五、链表 vs 数组（核心对比）
这是**面试必问**，必须牢记：

| 特性                | 数组                    | 链表                      |
|---------------------|-------------------------|---------------------------|
| 存储空间            | 连续                    | 不连续                    |
| 访问速度            | O(1) 随机访问           | O(n) 只能遍历访问         |
| 插入/删除（中间）| O(n) 需要移动元素       | O(1) 只需修改指针         |
| 内存大小            | 固定，不可动态扩展      | 动态，按需分配            |
| 额外开销            | 无                      | 每个节点带指针，开销更大  |
| 使用场景            | 频繁查询、少增删        | 频繁增删、少查询          |

**一句话总结：**
- 查得多 → 用**数组**
- 增删多 → 用**链表**

### 总结
1. **链表 = 节点 + 指针**，非连续存储，靠指针串联
2. **单链表**：简单省空间，只能向后遍历
3. **双链表**：支持前后遍历，插入删除更方便
4. **核心优势**：任意位置**插入/删除 O(1)**
5. **核心劣势**：查找慢，必须从头遍历
6. C++ 实现重点：**指针操作 + 内存管理（new/delete）**
