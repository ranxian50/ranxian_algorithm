# 一、什么是二叉树？
**二叉树 = 每个节点最多有 2 个子节点的树形结构**
- 最多两个孩子：**左孩子、右孩子**
- 子树有左右之分，顺序不能互换
- 是最基础、最重要的树形结构

形象理解：
像**家族树**，每个人最多两个孩子，左边是大儿子，右边是二女儿。

---

# 二、二叉树的基本结构
一个节点通常包含 3 部分：
1. **数据域**：存值
2. **左孩子指针**：指向左子树
3. **右孩子指针**：指向右子树

C++ 节点定义（最标准写法）：
```cpp
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x) : val(x), left(NULL), right(NULL) {}
};
```

---

# 三、二叉树的重要术语
- **根节点（Root）**：最顶层节点
- **叶子节点（Leaf）**：没有孩子的节点
- **父节点 / 子节点**
- **深度（Depth）**：从根到该节点的层数（根一般为 0 或 1）
- **高度（Height）**：从该节点到最远叶子的最长路径
- **子树**：以某个孩子为根的树

# 四、特殊二叉树（必须掌握）
## 1. 满二叉树
- 所有层都填满
- 每一个非叶子节点都有**两个孩子**
- 叶子全部在最底层

## 2. 完全二叉树
- 除最后一层外，其他层全满
- 最后一层节点**靠左连续排列**
- **堆（Heap）就是完全二叉树**

## 3. 二叉搜索树 BST（最常用）
规则：
- 左子树所有节点 **< 根节点**
- 右子树所有节点 **> 根节点**
- 左右子树也都是 BST

作用：**快速查找、插入、删除**

## 4. 平衡二叉树
左右子树高度差 ≤ 1，且左右子树都平衡。
保证效率：`O(log n)`

# 五、二叉树的重要性质（面试必考）
1. 第 i 层最多有 **2^(i-1)** 个节点
2. 高度 h 的树最多 **2^h - 1** 个节点
3. n 个节点的完全二叉树高度：**floor(log₂n) + 1**
4. n 个节点的二叉树，叶子数 = 度为 2 的节点数 + 1
5. 完全二叉树可用**数组**存储：
   - 下标 i 的左孩子：`2*i + 1`
   - 右孩子：`2*i + 2`
   - 父节点：`(i-1)/2`

# 六、二叉树的四种遍历（核心！）
## 1. 前序遍历
**根 → 左 → 右**

```cpp
void preOrder(TreeNode* root) {
    if (!root) return;
    cout << root->val;
    preOrder(root->left);
    preOrder(root->right);
}
```

## 2. 中序遍历
**左 → 根 → 右**
⚠️ BST 中序遍历 = **升序序列**

```cpp
void inOrder(TreeNode* root) {
    if (!root) return;
    inOrder(root->left);
    cout << root->val;
    inOrder(root->right);
}
```

## 3. 后序遍历
**左 → 右 → 根**

```cpp
void postOrder(TreeNode* root) {
    if (!root) return;
    postOrder(root->left);
    postOrder(root->right);
    cout << root->val;
}
```

## 4. 层序遍历（BFS）
从上到下、从左到右一层一层遍历
用 **队列** 实现

```cpp
void levelOrder(TreeNode* root) {
    queue<TreeNode*> q;
    q.push(root);
    while (!q.empty()) {
        TreeNode* cur = q.front(); q.pop();
        cout << cur->val;
        if (cur->left) q.push(cur->left);
        if (cur->right) q.push(cur->right);
    }
}
```

# 七、二叉树常见操作（必会）
## 1. 求树的高度
```cpp
int getHeight(TreeNode* root) {
    if (!root) return 0;
    return 1 + max(getHeight(root->left), getHeight(root->right));
}
```

## 2. 统计节点总数
```cpp
int countNodes(TreeNode* root) {
    if (!root) return 0;
    return 1 + countNodes(root->left) + countNodes(root->right);
}
```

## 3. 判断是否平衡二叉树
```cpp
bool isBalanced(TreeNode* root) {
    if (!root) return true;
    int lh = getHeight(root->left);
    int rh = getHeight(root->right);
    return abs(lh-rh) <= 1 
        && isBalanced(root->left) 
        && isBalanced(root->right);
}
```

## 4. 判断是否二叉搜索树
利用中序遍历是递增序列判断。

# 八、二叉树的应用（非常重要）
- **二叉搜索树**：快速查找、数据库索引
- **堆（完全二叉树）**：优先队列、堆排序、TopK
- **哈夫曼树**：数据压缩
- **红黑树 / AVL树**：平衡二叉树，STL map、set 底层
- **B+ 树**：MySQL 索引
- **表达式树**：编译原理计算表达式
