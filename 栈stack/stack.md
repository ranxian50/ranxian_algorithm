# 一、栈是什么？一句话定义
**栈是一种「后进先出」（Last In First Out，LIFO）的线性数据结构。**
你可以把它理解成：
- 一叠盘子
- 弹夹
- 浏览器的后退
- 函数调用栈

只能**从一头进、从同一头出**，另一头被封住。

---

# 二、栈的核心特点
1. **只有一端可以操作**，叫**栈顶（top）**
2. 另一端固定，叫**栈底（bottom）**
3. **只能在栈顶添加/删除**
4. 规则：**后放入的先取出**
5. 是一种**逻辑结构**，底层可用数组或链表实现

---

# 三、栈的基本操作（必考）
栈只有 4 个核心操作：

1. **push（入栈 / 压栈）**
   把元素放到栈顶。

2. **pop（出栈 / 弹栈）**
   把栈顶元素删除并返回。

3. **peek / top（取栈顶元素）**
   只看栈顶，不删除。

4. **isEmpty / isFull**
   判断栈空 / 栈满（顺序栈需要）

---

# 四、栈的两种实现方式
## 1. 顺序栈（用数组实现）
- 底层是数组
- 访问快、连续内存
- 有固定容量，满了不能再 push

## 2. 链式栈（用链表实现）
- 底层是单链表
- 容量理论无限
- 每个节点多存一个指针，略占内存

实际刷题、面试**优先用数组实现**，最简单高效。

# 五、栈的经典应用（超级重要）
栈是所有算法里**应用最广**的数据结构之一，下面这些你必须会：

## 1. 函数调用栈（程序运行的基础）
- 调用函数 → push
- 函数返回 → pop
递归本质就是栈。

## 2. 括号匹配（LeetCode 20）
`(){}[]` 是否合法，经典栈题。

## 3. 表达式求值
- 中缀表达式转后缀（逆波兰）
- 计算器原理

## 4. 浏览器前进后退
后退就是 pop，新页面就是 push。

## 5. 深度优先搜索 DFS
DFS 本质就是栈（递归/非递归都一样）。

## 6. 单调栈（面试高频）
解决：
- 下一个更大元素
- 接雨水
- 柱状图最大矩形
难度高，但非常高频。

# 六、入栈出栈序列规律（考试常考）
给定入栈顺序：`1 2 3 4 5`
下面哪些出栈顺序**合法**？

合法例子：
- 5 4 3 2 1
- 3 2 1 5 4
- 2 1 4 3 5

非法例子：
- 3 1 2 4 5（1 不可能在 2 前面出来）

**判断口诀：**
出栈序列中，**每个元素后面，比它小的元素必须严格递减**。

# 七、C++ 数组实现栈（极简可直接用）
```cpp
class Stack {
private:
    int arr[100];
    int top;

public:
    Stack() {
        top = -1;
    }

    // 入栈
    void push(int x) {
        if (top >= 99) return; // 栈满
        arr[++top] = x;
    }

    // 出栈
    void pop() {
        if (top < 0) return;
        top--;
    }

    // 取栈顶
    int peek() {
        return arr[top];
    }

    bool empty() {
        return top == -1;
    }
};
```

# 八、Java / Python 栈写法（快速参考）
## Java
```java
Stack<Integer> st = new Stack<>();
st.push(1);
st.pop();
st.peek();
st.isEmpty();
```

## Python
```python
st = []
st.append(1)  # push
st.pop()      # pop
st[-1]        # peek
len(st) == 0  # isEmpty
```

# 九、栈的时间复杂度
- **push、pop、peek：O(1)**
- 查找：O(n)（栈不适合查找）
